# SkyWay iOS-SDK

このリポジトリは、2023 年 1 月 31 日にリリースされた SkyWay の iOS SDK です。旧 SkyWay の iOS SDK とは互換性がありません。

## ドキュメント

### 公式サイト

https://skyway.ntt.com/ja/docs/user-guide/ios-sdk/

### API リファレンス

- [Room ライブラリ](https://ios-sdk.api-reference.skyway.ntt.com/room)

### 対応OS
iOS14+, iPadOS14+
